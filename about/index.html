前端作品

1. npm-get-weather

项目说明：基于Node的查询天气的命令行工具，可以通过命令行来查询当地或其他城市的天气。发布一个Node包首先注册npm账号，然后新建项目通过登录npm账号publish该项目即可，通过修改package.json，加入 bin 字段，即可开发命令行工具。

关键词： Node npm Promise等

项目预览： npm-get-weather

2. wheel-MVVM

项目说明：实现一个 微型的MVVM 框架，主要包括指令、单向和双向数据绑定。主要核心原理是两部分：数据监控和模板解析。通过Object.defineProperty实现对data数据的监听，而初始化View时涉及该数据的模板或指令进行解析处理；这里还涉及到观察者模式，主题（Subject）可以看做是Model里的数据，而观察者（Observer）就是View用到该数据的指令或模板，当数据一旦改变时就会通知所有订阅者更新。

关键词：JavaScript MVVM DOM等

项目预览： wheel-MVVM

3. node-server

项目说明：实现一个简易可扩展能处理路由解析和支持静态目录的 server。基本原理是基于Node的http模块创建HTTP服务器，根据http请求里的信息来进行相应的处理。

关键词：JavaScript Node HTTP等

查看源码： node-server

4. Music-FM

项目说明：Music-FM是一个在线音乐播放器，利用在线音乐API，可在线播放音乐，进行播放、暂定、下一曲和歌词查看等操作，在播放时歌词根据播放进度进行自动滚动，也可以选择单曲循环。

关键词：JavaScript DOM Ajax CSS等

项目预览： Music-FM

5. IFE(百度前端技术学院)项目集

项目说明：百度前端技术学院是一个免费的前端技术实践、分享、交流平台。通过精心编写了数十个实践编码任务，将技术知识点系统有机地串联在各个充满趣味与挑战的任务中，通过实际地编码练习来掌握知识点。

关键词：HTML、CSS、JavaScript、Node 常见布局 DOM 编码规范 Ajax 正则 this 事件模型等

项目预览： IFE-2017
